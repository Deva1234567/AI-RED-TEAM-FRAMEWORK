# AI Red Team Analyst Workbench

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![Streamlit](https://img.shields.io/badge/Streamlit-1.30+-red.svg)](https://streamlit.io/)

A complete **AI Vulnerability Assessment & Penetration Testing (VAPT)** platform for red teaming Large Language Models (LLMs). Built with a focus on **MITRE ATLAS**, **OWASP LLM Top 10**, and real-world adversarial testing workflows.

This tool enables security analysts to perform **automated red team campaigns**, **manual attack crafting**, **deep investigation**, and **executive-level reporting** ‚Äî all in a local, private, and offline-capable environment.

## Features

- **Automated Red Team Simulation**  
  Launch large-scale attack campaigns with configurable payload counts, mutation types (e.g., paraphrase, obfuscation), and defense hardening via system prompts.

- **Manual Attack Lab**  
  Craft custom payloads, mutate them, mark severity, and execute attacks interactively against any supported LLM.

- **Investigation Workspace**  
  Detailed case-by-case analysis with full request/response history, MITRE ATLAS classification, refusal quality scoring, and analyst notes.

- **Executive Report Builder**  
  Automated findings summary with total vulnerabilities, severity breakdown, resilience scoring, and JSON export for compliance.

- **Visual Analytics**  
  Interactive charts (bypass rate pie chart, timelines, severity distribution) for quick insights.

- **Local & Private Deployment**  
  Runs fully offline with local models (e.g., Ollama, Hugging Face Transformers). No data leaves your machine.
